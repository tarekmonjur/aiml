
### Use Virtual Env
- Create a new virtual event
  - **python -m venv venv**
- Active the Env
  - **venv\Scripts\activate** (windows)
  - **source venv/bin/activate** (linux)

### Package Install and Manage
- Install Package
  - **pip install package_name**
- Manage package lock
  - **pip freeze > requirements.txt**
  - **pip install -r requirements.txt**
